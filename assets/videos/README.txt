# Video Assets Directory

1.  **Drop your files here**: Place your `.mp4` video files in this folder.
2.  **Update Config**: Open `js/video.js` and update the `list` array to match your filenames.

Example:
If you add `my_vacation.mp4`, update `js/video.js`:
```javascript
App.Video = {
    list: ['my_vacation.mp4'],
    ...
}
```
